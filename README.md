# Block Saver App

A Tkinter-based Python app for splitting large text/code into blocks and reconstructing them. Supports raw text, JS, and JSON modes.

## Installation
1. Clone the repo: `git clone https://github.com/yourusername/block-saver-app.git`
2. Install deps: `pip install -r requirements.txt`
3. Run: `python src/main.py`

## Usage
- Split tab: Paste/load content, choose mode, prefix, output dir, and split.
- Reconstruct tab: Select input dir with blocks, prefix, output file, and reconstruct.

Features: Dark theme, file explorer, progress bars.
